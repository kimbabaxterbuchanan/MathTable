# MathTable
This application will allow a young student to practice basic Multiplication and Division Skills.  It has selections for Missing product, multiplcand, multiplier, divisor, dividend, qoutient and remainder.  It contains the Multiplication Table as well as Multiply and Divide Show Work section with speach. 

This program is standalone and requires no special libraries that are not included in .Net Framework 4.8.
To use this program 
1. Make selection of Multiply or Divide
2. Select the operation to be performed
3. Enter the missing value
4. Click on the Answer button
5. If answer is correct, select Next button to select next random values.
6. If having problems determining the correct answer, check the Show Answer checkbox to display the correct answer.

The Show Multiplication Table button will display a standalone form with the table populated for value of 1 to 12.

The Show Work buton will display a window popup in the basic format of a multiplication or division problem.  To use
this form:
Multiply
1. press the Clear button
2. Enter the Multiplicand value
3. Enter the Multiplier value
4. Press the Multiply button - this will show the steps to provide the correct answer.

Divide
1. press the Clear button
2. Enter the Divisor value
3. Enter the Dividend value
4. Press the Divide button - this will show the steps to provide the correct answer.

the Read button will activate the speech module to read the math problem to the student.  You can switch between a Male or Female voice
